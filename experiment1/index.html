Experiment 1 - Fish Name Generator

Description

This Experiment is the introduction to the course and allowed me to create a generator that creates a name for the type of fish it randomly chooses.

Technical

It uses a simple template system with placeholder tags like $fish and $fishname. When the reroll button is clicked, JavaScript replaces those placeholders with a random fish and a name matched to that fish using a mapping structure.

Reflection

This was a fun introduction to DOM manipulation and working with templates. It was also a nice introduction and refresher on JavaScript and HTML. It helped clarify the value of separating data from display logic.
